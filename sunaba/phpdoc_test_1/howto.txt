■ インストール
pear install --alldeps phpdocumentor

■ ドキュメントの生成方法（基本）
phpdoc -t {ドキュメントを生成するディレクトリ} -d {ソースディレクトリ} -o {出力用テンプレート}

■ ドキュメントの生成方法（Eclipse）
1. [実行] -> [外部ツール] -> [外部ツールの構成] から[プログラム]の部分に新しい構成を作成
2. ロケーションにはphpdocのスクリプトを指定
3. 作業用ディレクトリはドキュメントを生成したいディレクトリなど
4. 引数（作業用ディレクトリをjsdoc-toolkitに指定した場合）
	 -t {ドキュメントを生成するディレクトリ} -d {ソースディレクトリ}
5. 実行すると指定ディレクトリにドキュメントが作られる

■ 備考
デフォルトテンプレートだと日本語が文字化けするので修正が必要になる。
